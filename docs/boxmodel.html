box-sizing의 기본값은 content-box이고 padding, border-width는 width, height 계산에서 제외된다.

content-box
padding: 10px;
content-box
padding: 0;
border-box
padding: 10px;
border-box
padding: 0;

background-clip으로 background가 미치는 범위를 지정할 수 있다.

border-box
padding-box
content-box

background-origin은 background image의 시작 위치를 지정하게 한다. 디폴트는 padding-box이다. background-origin을 기준으로 표시하고 background-clip에 의해서 잘리는 것을 알 수 있다.

border-box
padding-box
content-box
border-box
padding-box
content-box

Containing box의 width가 기준일 경우에 containing box의 box-sizing은 어떤 영향을 미치는가? Containing box의 box-sizing이 border-box인 경우에는 width에서 border-width와 padding을 제외한 값이 기준이 된다.

border-box
width: 200px;
padding: 10% -> 14px;
content-box
width: 200px;
padding: 10% -> 20px;

Nested element의 margin은 containing box에 border, padding이 없으면 collapse 된다. top, bottom인 경우에만 해당한다.

Padding을 추가하면 collapse 되지 않는다.
